Tool 3 — Time-Dependent Data Plotting & Animation

Python toolkit for Excel-based time series with 2D/3D plotting, lag plots, and animation support.

What this toolkit does

  • Loads Excel data with one time column and multiple variables
  • Lets you choose a skip value, where 1 means no skipping
  • Supports standard 2D time-series plots and 3D trajectory plots
  • Supports 2D lag plots such as x(n) versus x(n+skip)
  • Allows custom axis labels, variable names, and titles for presentation-ready figures

Required Excel structure

Your sheet should contain one time column and at least three variable columns. Units can be included directly in the header names.

Time (s) Variable 1 (a.u.) Variable 2 (a.u.) Variable 3 (a.u.)
0.000.001.000.15
0.010.120.990.18
0.020.240.960.22
0.030.350.920.27

The downloadable workbook includes a full example sheet using this format.

Example usage

from toolkit3_time_dependent_plotting import *
from matplotlib.animation import PillowWriter

# Load the Excel workbook
# skip=1 means no skipped time steps
df = load_excel_timeseries(
    "toolkit3_example_timeseries.xlsx",
    sheet_name="Example_Data",
    time_col="Time (s)",
    variable_cols=["Variable 1 (a.u.)", "Variable 2 (a.u.)", "Variable 3 (a.u.)"],
    skip=1,
)

# Make a 2D lag plot: x(n) vs x(n+1)
plot_lag_2d(
    df,
    variable_col="Variable 1 (a.u.)",
    skip=1,
    xlabel="Var1(n)",
    ylabel="Var1(n+1)",
    title="2D lag plot",
)

# Build a 3D animation
anim = animate_timeseries(
    df,
    mode="3d",
    x_col="Variable 1 (a.u.)",
    y_col="Variable 2 (a.u.)",
    time_col="Time (s)",
    xlabel="Variable 1",
    ylabel="Variable 2",
    zlabel="Time (s)",
    title="3D animation",
)
anim.save("demo.gif", writer=PillowWriter(fps=15))

You can change the labels and choose different variables depending on the figure you want to generate.
