Stepwise GIF Builder for Time-Series Data

This version follows the same logic as your Jupyter workflow. First load the Excel or CSV file and detect the variables. Then choose whether the output frames should be 2D or 3D, choose whether you want a time-dependent map or a return map, export a ZIP of rendered frames, and finally upload that ZIP to create the GIF.
